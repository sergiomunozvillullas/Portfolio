Proyectos Académicos

DAW - Desarrollo de Aplicaciones Web

Realicé mis estudios en el IES Leonardo Da Vinci. Mi proyecto final fue una página web de un restaurante para hacer pedidos y reservas, integrando un método de pago con Redsys. La aplicación incluía:

  • Creación de cuentas con login y registro.
  • Menú interactivo con platos y menús según el día.
  • Reservas en el restaurante con disponibilidad de mesas.
  • Integración de pagos seguros.
  • Panel de administración con funcionalidades diferenciadas para administradores y clientes, con vistas personalizadas.
  • Diseño responsivo y backend utilizando HTML, CSS y JavaScript para la interfaz, y PHP con MySQL para gestionar la lógica y la base de datos, asegurando una experiencia fluida y eficiente.

También estudié en el IES Leonardo Da Vinci. Algunos de los proyectos que desarrollé incluyen:
